# Healthcare SQL Project

## Project Overview

This project analyzes Obsessive-Compulsive Disorder (OCD) patient data using SQL.

The goal is to explore patient demographics, symptom severity, diagnosis patterns, and treatment-related insights through SQL queries.

## Dataset

The dataset contains information about OCD patients, including:

- Patient ID
- Gender
- Age
- Ethnicity
- Obsession Type
- Compulsion Type
- Y-BOCS Obsession Score
- Y-BOCS Compulsion Score
- Diagnosis Date

## Tools Used

- MySQL
- SQL
- GitHub
